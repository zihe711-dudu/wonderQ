# ⚡ 快速啟動指南

## 最簡單的方式（3 步）

### 方法 1：使用啟動腳本（推薦）

**macOS/Linux：**
```bash
./start.sh
```

**Windows：**
```bash
start.bat
```

或雙擊 `start.sh`（macOS/Linux）或 `start.bat`（Windows）文件

---

### 方法 2：手動啟動

```bash
# 1. 進入專案目錄
cd quizy-tw-kids

# 2. 安裝依賴（首次需要）
npm install

# 3. 啟動服務器
npm run dev
```

然後訪問：**http://localhost:3000**
